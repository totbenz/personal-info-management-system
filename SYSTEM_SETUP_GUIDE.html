Personal Info Management System

Complete Setup Guide

This guide will help you set up the Personal Info Management System from scratch on a new laptop/computer.

System Requirements

Hardware Requirements

  • RAM: Minimum 8GB, Recommended 16GB or higher
  • Storage: Minimum 10GB free disk space
  • Processor: Modern dual-core processor or better

Software Requirements

1. PHP

  • Version: PHP 8.1 or higher (8.2 recommended)
  • Required Extensions:
php-cli
php-fpm
php-mysql (or php-pgsql for PostgreSQL)
php-xml
php-curl
php-zip
php-gd
php-mbstring
php-bcmath
php-json
php-tokenizer
php-ctype
php-fileinfo
php-opcache (recommended for performance)

2. Web Server

  • Apache 2.4+ with mod_rewrite enabled OR
  • Nginx 1.18+ with proper rewrite rules OR
  • Use PHP's built-in development server (for development only)

3. Database

  • MySQL 5.7+ or MariaDB 10.3+ OR
  • PostgreSQL 12+ OR
  • SQLite 3.8+ (for development)

4. Node.js & NPM

  • Node.js: Version 18.x or 20.x (LTS versions recommended)
  • NPM: Version 9.x or 10.x (comes with Node.js)

5. Composer

  • Composer: Version 2.0 or higher

6. Git

  • Git: Version 2.30 or higher (for version control)

Installation Steps

Recommended Approach: This guide uses Laragon for Windows, which provides an all-in-one development environment with PHP, MySQL, Apache, and more.

Windows Installation with Laragon

Step 1: Install Laragon

Laragon is a powerful development environment for Windows that includes everything you need:

  • PHP 8.2 (latest version)
  • MySQL & MariaDB
  • Apache Server
  • Redis
  • phpMyAdmin
  • Composer pre-installed
  • And more...
Download Laragon:
  1. Visit https://laragon.org/download/
  2. Download Laragon Universal (recommended) or Laragon Full
  3. Run the installer as Administrator
  4. Follow the installation wizard (default settings are fine)
  5. Launch Laragon when installation completes
Important: Run Laragon as Administrator for the first time to ensure all services start properly.

Step 2: Verify Laragon Installation

Start all services in Laragon:

  1. Open Laragon
  2. Click "Start All" button (or press Ctrl+Alt+S)
  3. Wait for all services to turn green
  4. Test by visiting http://localhost - you should see Laragon welcome page
  5. Access phpMyAdmin at http://localhost/phpmyadmin

Step 3: Install Git

# Download and install from https://git-scm.com/download/win
# During installation, use recommended settings
# Verify installation:
git --version

Step 4: Install GitHub Desktop (Optional but Recommended)

GitHub Desktop provides a user-friendly interface for managing Git repositories:

  1. Download from https://desktop.github.com/
  2. Install and sign in with your GitHub account
  3. Configure your Git credentials if prompted

Step 5: Verify Composer (Pre-installed in Laragon)

Laragon comes with Composer pre-installed. To verify:

# Open Laragon Terminal (Menu > Laragon > Terminal)
composer --version

Step 6: Install Node.js

# Download and install from https://nodejs.org/
# Choose the LTS version (Long Term Support)
# Verify installation:
node --version
npm --version

Step 7: Install Laravel (Global Install)

Install Laravel installer globally for easier project creation:

# In Laragon Terminal
composer global require laravel/installer

macOS Installation

1 Install Homebrew (if not installed) 
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
2 Install PHP, Composer, Node.js, and MySQL 
brew install php composer node mysql
3 Start MySQL 
brew services start mysql

Linux (Ubuntu/Debian) Installation

1 Update System 
sudo apt update && sudo apt upgrade -y
2 Install PHP and Extensions 
sudo apt install php8.2 php8.2-fpm php8.2-mysql php8.2-xml php8.2-curl php8.2-zip php8.2-gd php8.2-mbstring php8.2-bcmath php8.2-json php8.2-tokenizer php8.2-ctype php8.2-fileinfo php8.2-opcache -y
3 Install Composer 
curl -sS https://getcomposer.org/installer | php
sudo mv composer.phar /usr/local/bin/composer
sudo chmod +x /usr/local/bin/composer
4 Install Node.js 
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install nodejs -y
5 Install MySQL 
sudo apt install mysql-server -y
sudo mysql_secure_installation
6 Install Git 
sudo apt install git -y

Step 8: Clone the Project

Now that all prerequisites are installed, let's clone the project:

Option 1: Using Git Command (in Laragon Terminal)

# Navigate to Laragon www directory
cd C:\laragon\www

# Clone the repository
git clone https://github.com/totbenz/personal-info-management-system.git

cd personal-info-management-system

  1. Open GitHub Desktop
  2. Click File > Clone Repository
  3. Enter URL: https://github.com/totbenz/personal-info-management-system.git
  4. Choose local path: C:\laragon\www\personal-info-management-system

Common Issues & Solutions

1. Composer Memory Limit Error

composer install --memory-limit=2G
# or
php -d memory_limit=2G /usr/local/bin/composer install

2. Node.js Version Mismatch

# Install specific Node version using nvm
nvm install 20
nvm use 20

3. Database Connection Error

  • Verify MySQL/MariaDB service is running
  • Check database credentials in .env
  • Ensure database exists

4. Permission Issues (Linux/macOS)

sudo chmod -R 775 storage bootstrap/cache
sudo chown -R www-data:www-data storage bootstrap/cache  # for web server user

5. Vite Build Issues

# Clear cache and rebuild
npm run build -- --force
# or
rm -rf node_modules package-lock.json
npm install
npm run build

6. Livewire Not Working

  • Ensure assets are built: npm run dev
  • Check that @livewireStyles and @livewireScripts are in your layout
  • Verify routes are cached: php artisan route:clear

Production Deployment

1. Optimize for Production

# Install production dependencies
composer install --optimize-autoloader --no-dev

# Cache configuration
php artisan config:cache
php artisan route:cache
php artisan view:cache

# Build assets for production
npm run build -- --production

2. Set Up Cron Jobs

# Edit crontab
crontab -e

# Add Laravel scheduler
* * * * * cd /path-to-your-project && php artisan schedule:run >> /dev/null 2>&1

3. Queue Worker (if using queues)

# Install supervisor to manage queue workers
sudo apt install supervisor

# Configure supervisor config in /etc/supervisor/conf.d/laravel-worker.conf

Security Considerations

  1. Never commit .env file to version control
  2. Use HTTPS in production
  3. Set APP_DEBUG=false in production
  4. Regularly update dependencies 
    composer update
npm update
  5. Use strong passwords for database
  6. Enable firewall on production server
  7. Regular backups of database and files

Support & Troubleshooting

If you encounter issues:

  1. Check Laravel logs: storage/logs/laravel.log
  2. Enable debug mode temporarily: APP_DEBUG=true in .env
  3. Check PHP error logs
  4. Verify all requirements are met
  5. Check official documentation:
    • Laravel: https://laravel.com/docs
    • Livewire: https://livewire.laravel.com/docs
    • Tailwind CSS: https://tailwindcss.com/docs

Complete Quick Start Summary

Here's the complete command sequence for setting up the project from scratch with Laragon:

# 1. Navigate to Laragon www directory
cd C:\laragon\www

# 2. Clone the project
git clone https://github.com/totbenz/personal-info-management-system.git
cd personal-info-management-system

# 3. Install dependencies
composer install
npm install

# 4. Setup environment
cp .env.example .env
php artisan key:generate

# 5. Create database (manually in phpMyAdmin: "personal_info_system")

# 6. Run migrations and seed
php artisan migrate:fresh --seed

# 7. Build assets
npm run dev

# 8. Visit your application
# http://localhost/personal-info-management-system
That's it! Your Personal Info Management System is now running. No need to run php artisan serve with Laragon - it's automatically configured to serve your project.
